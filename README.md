# NOVA

NOVA is a Llama-powered AI assistant focused on coding, learning, career growth, creativity, and personalized conversations.

## Features

- Coding Support
- Learning Guidance
- Career Guidance
- Creative Writing
- Emotional Support
- Knowledge Sharing
- Language Support
- Personalized Responses

## Technologies Used

- HTML
- CSS
- JavaScript
- Groq API
- Llama

## How to Use

1. Open NOVA.
2. Enter your Groq API key in Settings.
3. Start chatting.
